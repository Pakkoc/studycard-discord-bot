# discord 개발자 권한 설정
## 1. Application 생성
    새 Application 생성 → 아이콘/이름 설정

## 2. Bot 추가 및 기본 옵션
    Bot 탭 → Add Bot
    Token 생성 후 보관(.env의 DISCORD_BOT_TOKEN)
    Public Bot: 내부 전용이면 Off 권장, 여러 서버에서 쓸 계획이면 On
    Require OAuth2 Code Grant: Off

## 3. Privileged Gateway Intents
    Bot 탭 → Privileged Gateway Intents
    Server Members Intent: On (멤버 목록, 닉변, 입퇴장 처리에 필요)
    Message Content Intent: On (특정 채널 글 작성 시 XP 지급 로직에 필요)
    Presence Intent: Off (이 프로젝트에서는 미사용)

## 4. 설치(초대) 링크 구성
    Installation 또는 OAuth2 → Install Link(URL Generator)
    Scopes:
      applications.commands
      bot
    
    Bot Permissions(최소 권한 권장):
      View Channels
      Send Messages
      Send Messages in Threads
      Attach Files
      Embed Links

    Read Message History (권장)
      불필요: Manage Messages, Manage Channels, Connect/Speak 등은 요구되지 않음
      생성된 링크로 서버에 초대
